# Impeccable Design Context

> 由 teach-impeccable 流程于 2026-07-22 建立。后续所有 UI 设计工作以本文件为准。

## Design Context

### Users
- **双层用户**：命理爱好者/大众用户 + 专业命理师/深度研究者。
- 信息架构必须**渐进式披露**：默认简洁视图（爱好者友好，重引导与解释），专业数据（藏干、纳音、星曜亮度、四化飞星等）可一键展开，效率不受损。
- 使用情境：静心研究、排盘比对、AI 辅助解读——界面应让人「沉下心来」，而非催促操作。

### Brand Personality
- 三个词：**温润、克制、有根**。
- 情绪目标：翻宋版书般的安定感与敬意；数据精确带来的信任感。
- 反参照：科技仪表盘/终端风、赛博玄学（霓虹+深空+发光）、俗艳的「算命网站」红黄堆砌。

### Aesthetic Direction
- **宋韵极简·暖纸光**（浅色主题）：宣纸暖白底、墨色文字、朱砂点睛、鎏金/黛蓝/松烟绿少量辅助。
- 宋版书语言：细线（hairline）分隔、方角或微圆角、留白为奢、竖排汉字作装饰性标签、印章为品牌印记。
- 字体：标题宋体衬线（Noto Serif SC / Songti SC），正文黑体（Noto Sans SC / PingFang SC），数字用衬线 + tabular-nums。
- 色彩用 OKLCH 定义；中性色带暖调；禁纯黑 #000 / 纯白 #fff。

### Design Principles
1. **纸为底，墨为骨**：一切界面元素像落在宣纸上的墨迹——细线、留白、无厚重投影与玻璃拟态。
2. **朱砂不过三**：强调色（朱砂红）单屏出现不超过三处，用于最关键的操作与标记；滥用即失效。
3. **简繁两便**：每个视图都有「爱好者层」与「专业层」，默认简，展开即繁，两层数据同源。
4. **字即装饰**：依靠衬线汉字排版、竖排、印章、卷次编号营造东方气质，不依赖图标堆砌与插画。
5. **静而微动**：动效仅服务状态变化（展开、选中、切换），指数缓出，150–250ms，无回弹、无装饰性漂浮动画。
