## 一、核心概念

### 基本教义（原始佛教）

**四圣谛**：
1. **苦谛**：人生本质是苦（生老病死、爱别离、怨憎会、求不得、五蕴炽盛）
2. **集谛**：苦的原因——贪嗔痴三毒
3. **灭谛**：苦可以止息——涅槃
4. **道谛**：止息苦的道路——八正道

**八正道**：
- 正见、正思惟（智慧）
- 正语、正业、正命（戒律）
- 正精进、正念、正定（禅定）

### 中国佛教特色概念

- **因果业力**：善有善报，恶有恶报，不是不报，时候未到
- **缘起性空**：一切事物因缘和合而生，没有独立自性
- **无常**：一切都在变化中，没有永恒不变的事物
- **无我**：没有固定不变的自我实体
- **中道**：不偏不倚，不走极端

## 二、心性调适方法

### 正念禅修（适用于日常）

**基本方法**：
1. **呼吸觉察**：专注呼吸，念头来去不理
2. **身体扫描**：从头到脚觉察身体感受
3. **行禅**：行走中保持觉知
4. **日常正念**：吃饭、洗碗、走路时保持当下觉知

**咨询应用**：
- 焦虑→正念呼吸，回到当下
- 失眠→身体扫描放松
- 情绪波动→觉察情绪而不被带走

### 慈心禅（适用于人际困扰）

- 对自己：愿我快乐、平安
- 对亲近者：愿你快乐、平安
- 对中性人：愿你快乐、平安
- 对困难者：愿你快乐、平安
- 对一切众生：愿一切众生快乐、平安

### 因果思惟（适用于面对困境）

- 正视当前的果报是过去种因的结果
- 当下种善因，未来得善果
- 消业心态：以积极态度面对困难，视为消业和成长的机会

## 三、佛教与心理健康的融合

### 现代应用

| 问题 | 佛教视角 | 调适方法 |
|------|---------|---------|
| 焦虑 | 执着未来，心不安住当下 | 正念禅修、回到当下 |
| 抑郁 | 执着过去，被悔恨困住 | 接纳现实、慈悲自己 |
| 愤怒 | 嗔恨心起，烧毁功德林 | 慈心禅、观想 |
| 贪欲 | 不知足，欲壑难填 | 知足感恩、少欲知足 |
| 傲慢 | 我执强大，轻视他人 | 无我观、平等心 |
| 嫉妒 | 见不得他人好 | 随喜赞叹 |

### 情志与心性的关系

从佛教视角看，情志问题根源在于三毒：
- **贪**→过度追求→焦虑、失落、不满
- **嗔**→厌恶对抗→愤怒、怨恨、烦躁
- **痴**→无明执著→困惑、偏见、固執

## 四、中国佛教宗派简介

### 禅宗

- **不立文字，直指人心，见性成佛**
- **六祖慧能**《坛经》："本来无一物，何处惹尘埃"
- 修行方法：参话头、默照禅
- **咨询应用**：放下概念执著，直接体认本心

### 净土宗

- 持名念佛，往生净土
- **修行方法**：专念"阿弥陀佛"
- **咨询应用**：简单重复的功课可以安定心神

### 唯识宗

- 万法唯识，三界唯心
- **八识**：前五识+意识+末那识+阿赖耶识
- **咨询应用**：认识自己的深层心识模式

## 六、佛教与道家的融合

| 概念 | 道家 | 佛教 |
|------|------|------|
| 终极 | 道 | 真如/佛性 |
| 方法 | 无为 | 无执 |
| 境界 | 天人合一 | 明心见性 |
| 对治 | 欲望→少私寡欲 | 执着→放下 |
| 态度 | 顺其自然 | 随缘不变 |

**融合应用**：
- 道家"道法自然" = 佛教"随缘"
- 道家"无为" = 佛教"不执着"
- 道家"齐物" = 佛教"平等心"
