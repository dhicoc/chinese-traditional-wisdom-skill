## 集成方式

### 方式一：作为 AI Skill 引用（推荐）

meihua-yishu 本身是 SKILL.md 格式的 AI skill，可直接作为知识参考：

```bash
git clone https://github.com/muyen/meihua-yishu.git
```

克隆后，将其 `SKILL.md` 和 `references/` 目录中的知识文件作为梅花易数分析的参考依据。当用户需要梅花易数占断时，按本文件的方法论进行解读。

### 方式二：Python 脚本调用

meihua-yishu 提供 `scripts/meihua_calc.py` 命令行工具：

```bash
# 时间起卦
python scripts/meihua_calc.py time 2026 6 28 14 30

# 时辰起卦（只到时辰精度）
python scripts/meihua_calc.py time-shichen 2026 6 28 14

# 数字起卦
python scripts/meihua_calc.py num 3 8
```

输出包含本卦、互卦、变卦、错卦、综卦、体用关系、卦气分析等结构化数据。

## 梅花易数分析框架

### 起卦方法

| 方法 | 输入 | 适用场景 |
|------|------|---------|
| 时间起卦 | 年月日时分（秒级） | 通用，最常用 |
| 数字起卦 | 两个数字 | 用户心中有数时 |
| 声音起卦 | 声音次数/特征 | 听到异常声响时 |
| 颜色起卦 | 颜色对应八卦 | 看到显著颜色时 |
| 测量起卦 | 测量数值 | 物体尺寸/数量 |
| 方位起卦 | 方位对应八卦 | 方位感知时 |

### 体用分析

梅花易数核心是体用关系：

**体卦**：代表自己/主体，卦中不变的卦
**用卦**：代表对方/客体/所测之事，卦中变化的卦

**五行生克关系**：

| 关系 | 含义 | 吉凶 |
|------|------|------|
| 用生体 | 外力帮助自己 | 吉 |
| 体生用 | 自己消耗精力 | 小凶 |
| 用克体 | 外力压制自己 | 凶 |
| 体克用 | 自己控制局面 | 吉 |
| 体用比和 | 双方和谐 | 吉 |

### 卦德分析（第二视角）

除五行生克外，还需考虑卦德：

| 八卦 | 卦德 | 方向 | 季节 |
|------|------|------|------|
| 乾 | 健（刚健） | 西北 | 秋冬间 |
| 兑 | 悦（喜悦） | 西 | 秋 |
| 离 | 丽（附丽） | 南 | 夏 |
| 震 | 动（行动） | 东 | 春 |
| 巽 | 入（进入） | 东南 | 春夏间 |
| 坎 | 陷（险陷） | 北 | 冬 |
| 艮 | 止（静止） | 东北 | 冬春间 |
| 坤 | 顺（柔顺） | 西南 | 夏秋间 |

### 关系场

- **互卦**：从本卦中提取内在因素，反映事物发展过程中的隐藏力量
- **变卦**：变爻后的卦，反映事物发展的最终走向
- **错卦**：阴阳全反的卦，反映事物的对立面
- **综卦**：上下颠倒的卦，反映换位思考的视角

## 在咨询中的使用

### 事业方向（与六爻互补）

1. 时间起卦（当前时间）
2. 确定体卦（自己）和用卦（事业环境）
3. 五行生克判断大势：用生体=有利，用克体=困难
4. 卦德看行业匹配：乾=金属/金融，离=IT/能源，震=木材/教育...
5. 变卦看发展趋势
6. 卦气看当前时机是否当令

### 婚恋趋势

1. 时间起卦或数字起卦
2. 体卦为自己，用卦为对方/感情
3. 体用比和=和谐，用生体=对方有意
4. 互卦看内在因素，变卦看发展走向
5. 注意：梅花易数看趋势，具体匹配还需八字合婚

### 心灵困惑

1. 梅花易数擅长"直读卦象"，适合直击问题本质
2. 错卦看对立面，综卦换位思考
3. 卦德提供哲学指导：如得艮卦则"止"——适时休息
4. 卦气判断当前是否当令，不当令则需等待时机
