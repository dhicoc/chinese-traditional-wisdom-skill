# 体质辨识问卷

> 本文件提供九种体质的标准化问卷，用于"中医"维度的体质辨识。基于中华中医药学会《中医体质分类与判定》标准。

---

## 使用方法

1. 逐一询问以下问题（或请用户自评）
2. 每题按 1-5 分评分：1=没有，2=很少，3=有时，4=经常，5=总是
3. 计算各体质类型总分
4. 最高分体质为主体质，次高分兼顾
5. 结果用于咨询中的个性化调养建议

---

## 问卷题目

### 气虚质（9题）

1. 您容易疲乏吗？
2. 您容易气短，呼吸短促吗？
3. 您容易心慌吗？
4. 您容易头晕或站起时晕眩吗？
5. 您比别人容易患感冒吗？
6. 您喜欢安静、懒得说话吗？
7. 您说话声音低弱无力吗？
8. 您活动量稍大就容易出虚汗吗？
9. 您容易出虚汗吗？

### 阳虚质（7题）

1. 您手脚发凉吗？
2. 您胃脘部、背部或腰膝部怕冷吗？
3. 您感到怕冷、衣服比别人穿得多吗？
4. 您比一般人耐受不了寒冷吗？
5. 您比别人容易患感冒吗？
6. 您吃（喝）凉的东西容易腹泻吗？
7. 您受凉或吃（喝）凉的东西后，容易腹泻吗？

### 阴虚质（8题）

1. 您感到手脚心发热吗？
2. 您感觉身体、脸上发热吗？
3. 您皮肤或口唇干吗？
4. 您口唇的颜色比一般人红吗？
5. 您容易便秘或大便干燥吗？
6. 您面部两颧潮红或偏红吗？
7. 您感到眼睛干涩吗？
8. 您感到口干咽燥，总想喝水吗？

### 痰湿质（8题）

1. 您感到胸闷或腹部胀满吗？
2. 您感到身体沉重不轻松或不爽快吗？
3. 您腹部肥满松软吗？
4. 您有额部油脂分泌多的现象吗？
5. 您上眼睑比别人肿（轻微隆起）吗？
6. 您嘴里有黏黏的感觉吗？
7. 您平时痰多，特别是咽喉部总感到有痰堵着吗？
8. 您舌苔厚腻或有舌苔厚厚的感觉吗？

### 湿热质（6题）

1. 您面部或鼻部有油腻感或者油亮发光吗？
2. 您易生痤疮或疮疖吗？
3. 您感到口苦或嘴里有异味吗？
4. 您大便黏滞不爽、有解不尽的感觉吗？
5. 您小便时尿道有发热感、尿色浓吗？
6. 您带下色黄（白带颜色偏黄）或阴部潮湿吗？（限女性回答）

### 血瘀质（7题）

1. 您皮肤在不知不觉中会出现青紫瘀斑吗？
2. 您两颧部有细微红丝吗？
3. 您身体上有哪里疼痛吗？
4. 您面色晦暗或容易出现褐斑吗？
5. 您容易有黑眼圈吗？
6. 您容易忘事（健忘）吗？
7. 您口唇颜色偏暗吗？

### 气郁质（7题）

1. 您感到闷闷不乐、情绪低沉吗？
2. 您容易精神紧张、焦虑不安吗？
3. 您多愁善感、感情脆弱吗？
4. 您容易感到害怕或受惊吓吗？
5. 您胁肋部或乳房胀痛吗？
6. 您无缘无故叹气吗？
7. 您咽喉部有异物感、且吐之不出咽之不下吗？

### 特禀质（7题）

1. 您没有感冒时也会打喷嚏吗？
2. 您没有感冒时也会鼻塞、流鼻涕吗？
3. 您有因过敏导致皮肤起包块（荨麻疹）吗？
4. 您皮肤因过敏出现过紫癜（紫红色瘀斑）吗？
5. 您皮肤一抓就红，并出现抓痕吗？
6. 您容易患哮喘吗？
7. 您容易对药物、食物、气味、花粉过敏吗？

### 平和质（8题，正向评分）

1. 您精力充沛吗？
2. 您容易疲乏吗？（反向计分）
3. 您说话声音低弱无力吗？（反向计分）
4. 您感到闷闷不乐吗？（反向计分）
5. 您比一般人耐受不了寒冷吗？（反向计分）
6. 您能适应外界自然和社会环境的变化吗？
7. 您容易失眠吗？（反向计分）
8. 您容易忘事（健忘）吗？（反向计分）

## 评分与判定

### 原始分转换

每体质类型：原始分 = 各题得分之和 / 题数 × 4 / 5

转换分范围：0-100 分

### 判定标准

| 转换分 | 判定 |
|--------|------|
| ≥60分 | "是"（该体质偏颇明显） |
| 40-59分 | "倾向是"（有该体质倾向） |
| <40分 | "否"（无明显该体质偏颇） |

### 平和质特殊判定

- 平和质转换分 ≥60 且其他8种偏颇体质转换分均 <40 → 判定为平和质
- 平和质转换分 ≥60 但某偏颇体质 ≥40 → 平和质兼夹该偏颇体质

## 体质与调理方向

| 体质 | 调养方向 | 食疗参考 | 穴位保健 |
|------|---------|---------|---------|
| 气虚质 | 补气健脾 | 黄芪、党参、山药 | 足三里、气海 |
| 阳虚质 | 温阳祛寒 | 生姜、肉桂、羊肉 | 关元、命门 |
| 阴虚质 | 滋阴清热 | 百合、麦冬、银耳 | 三阴交、太溪 |
| 痰湿质 | 化痰祛湿 | 陈皮、薏米、冬瓜 | 丰隆、阴陵泉 |
| 湿热质 | 清热利湿 | 绿豆、苦瓜、薏米 | 阴陵泉、曲池 |
| 血瘀质 | 活血化瘀 | 山楂、红花、桃仁 | 血海、膈俞 |
| 气郁质 | 疏肝理气 | 玫瑰花、合欢花、佛手 | 太冲、期门 |
| 特禀质 | 固表防敏 | 黄芪、白术、防风 | 足三里、风池 |
| 平和质 | 维持平衡 | 均衡饮食 | 常规保健 |

## 在咨询中的使用

1. 用户口述症状时，可快速对照问卷判断体质倾向
2. 不必逐题询问，可根据用户描述的症状直接判断
3. 体质结果与五运六气结合：当年岁运加重的体质偏颇需重点调养
4. 体质结果与命理结合：八字五行偏颇与体质有对应关系（如八字金旺多见肺系问题，对应气虚/阴虚质）
5. 所有食疗建议必须标注"仅供参考，具体请咨询中医师"

---

## 注意事项

- 体质判定需综合判断，单一症状不能确定体质
- 多数人为兼夹体质（2-3种混合），以偏颇最明显者为主
- 体质可随季节、年龄、环境变化而转变
- 本问卷为参考工具，不能替代中医师的当面辨证
- 女性经期、孕期体质会有暂时性变化，应标注特殊状态
