# 风水（堪舆）分析引擎集成指南

> 本文件指导如何在"择居选址"场景中使用风水 RAG 知识库进行分析。风水没有统一的开源排盘引擎，本 Skill 采用 RAG（Retrieval-Augmented Generation）+ 经典古籍的方式提供知识支撑。

---

## 引擎定位

**核心能力来源**：`knowledge-base/fengshui/` 目录下的 30 个古籍全文 + `mappings/` 目录下的 6 个确定性键值映射表

**不依赖外部 API / 不需要 RAG 和 embedding 模型**：所有知识来自本地结构化映射表和古籍文本，离线可用

**三个流派全覆盖**：形势派（峦头）、理气派（玄空/三元）、阳宅派（八宅/三要）

**双模式知识获取**：需要快速查表（命卦、方位、吉凶星）→ 读 mappings/ JSON；需要深度原文佐证 → 读古籍全文

**跨学科交叉**：风水与命理（八字喜用神配合坐向）、与五运六气（流年飞星）

## 分析工作流

### 第一步：输入收集

根据场景收集不同类型的信息：

**阳宅（住宅/办公室）需要**：
- 住宅坐向（如：子山午向、坐北朝南）
- 房屋格局描述（几室几厅、楼层、各房间位置）
- 大门位置、主卧位置、厨房位置
- 宅主八字（可选，用于配合喜用神）
- 周围环境（路、水、建筑、山形）
- 流年信息（可选，用于年命飞星分析）

**阴宅（祖坟/墓地）需要**：
- 坐向和分金（如有）
- 来龙去脉描述
- 周围砂水环境

### 第二步：知识检索

知识检索分两步进行：先查 `mappings/` 结构化映射表获取快捷数据，再读古籍全文获取经典原文佐证。

#### 2a. 确定性映射查找（优先）

在读取古籍全文之前，优先使用 `mappings/` 目录下的 JSON 文件进行快速结构化查询。这种方式无需 RAG 和 embedding 模型，直接通过键值匹配获取准确数据：

- **需要判断宅主命卦** → 读取 `mappings/life-trigram.json`，按出生年公式计算命卦（男：(100-年尾两位)%9，女：(年尾两位-4)%9，余0变9，余5男寄坤2女寄艮8），查东/西四命、吉凶方向
- **需要八宅方位吉凶** → 读取 `mappings/eight-mansions.json`，按宅主命卦×方位查64组吉凶星和含义（生气/天医/延年/伏位/绝命/五鬼/六煞/祸害）
- **需要二十四山向定位** → 读取 `mappings/twenty-four-mountains.json`，根据罗盘度数查山向五行/阴阳/三合/八卦归属
- **需要流年飞星** → 读取 `mappings/yearly-flying-stars.json`，查某年入中星、九宫飞布方位、五黄/二黑煞位
- **需要阳宅三要断语** → 读取 `mappings/three-essentials.json`，查门主灶关键组合吉凶和灶位规则
- **需要形煞化解方法** → 读取 `mappings/form-sha-cures.json`，查19种常见形煞的辨别标准、严重程度和化解方案

查表得出初步结论后，进入古籍全文检索获取经典原文佐证。

#### 2b. 古籍全文检索

完成映射查表后，根据需求读取对应的古籍知识库文件：

**形势分析** → 读取 01-situation-form/ 中相关经典：
- 读取葬书了解"生气"理论基础
- 读取撼龙经了解龙脉判断方法
- 读取雪心赋参考形势描述语汇

**理气分析** → 读取 02-principle-form/ 中相关经典：
- 读取天玉经/催官篇了解理气原则
- 结合流年推算九星飞布

**阳宅布局** → 读取 03-yang-house/ 中相关经典：
- 读取阳宅三要判断门主灶关系
- 读取阳宅十书参考具体布局规则
- 读取八宅明镜参考东西四命分配

### 第三步：多派别交叉验证

风水流派之间的观点可能有所差异，分析时应：

1. **形势优先**：先验证外部环境（山水格局）是否合理
2. **理气跟进**：再考虑坐向卦线和流年飞星
3. **阳宅为主**：对于住宅问题，阳宅三要/十书的实用性最高
4. **交叉验证**：三种派别看法一致则结论较稳，不一致时给出权衡建议

### 第四步：命理配合（可选）

如有宅主八字信息：
- 根据八字喜用神（金木水火土）选择住宅朝向和颜色
- 大运转换时相应调整居住环境
- 阳宅三要的门主灶配合宅主年命

### 第五步：输出建议

按模板 `templates/fengshui-consultation.md` 输出：
- 基本信息（坐向/布局/环境）
- 形势分析（龙穴砂水评估）
- 理气分析（玄空飞星/八宅游年）
- 命理配合（喜用神与布局建议）
- 具体调整建议（吉位催旺/煞位化解）
- RULES.md 合规（积极导向、不绝对化吉凶）

## 禁忌与合规

1. **不绝对化吉凶**：风水建议使用"宜/不宜"、"倾向"等措辞，而非"必吉/必凶"
2. **尊重隐私**：不要求上传住宅真实照片（可描述）
3. **不涉鬼神**：从环境心理学和传统文化角度解释，不做超自然解读
4. **安全优先**：涉及装修/改造的实质性建议需提醒咨询专业人士
5. **文化定位**：强调风水作为传统环境学的文化价值，预防医/建筑红线
