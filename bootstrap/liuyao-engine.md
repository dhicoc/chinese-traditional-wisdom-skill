# 六爻卜卦引擎集成指南

> 本文件指导如何集成 ichingshifa Python 库，用于"卜"维度的周易六爻卜卦。

---

## 引擎概览

**PyPI 包**：https://pypi.org/project/ichingshifa/

**版本**：3.1.9

**语言**：Python

**许可**：MIT

**安装**：

```bash
pip install ichingshifa sxtwl ephem numpy cn2an
```

**核心能力**：
- 大衍之数筮法（模拟 50 蓍草传统起卦流程）
- 本卦 + 变卦生成
- 朱熹《易学启蒙》变爻规则（0-6 爻全场景覆盖）
- 纳甲法（京房）：天干/地支/五行/六亲赋值
- 六神安放（青龙/朱雀/勾陈/螣蛇/白虎/玄武）
- 世应定位
- 伏神/藏干推演
- 完整排盘输出（结构化数据 + 文本面板）

## 集成方式

### 直接调用（推荐）

```python
from ichingshifa import ichingshifa

# 方式一：随机起卦
result = ichingshifa.ichingshifa()

# 方式二：指定时间起卦
from datetime import datetime
result = ichingshifa.ichingshifa(datetime(2026, 6, 28, 14, 30))

# 方式三：手动指定爻值（铜钱法）
# 6=老阴(变), 7=少阳, 8=少阴, 9=老阳(变)
result = ichingshifa.mget_bookgua_details('789789')  # 从初爻到上爻
```

## 六爻断卦框架

### 用神选取

根据占问事项确定用神：

| 占问事项 | 用神 | 六亲 |
|---------|------|------|
| 自己/求测人 | 世爻 | — |
| 他人 | 应爻 | — |
| 父母/长辈/文书/房屋 | 父母爻 | 生我者 |
| 兄弟/朋友/竞争 | 兄弟爻 | 同我者 |
| 子女/下属/宠物/药 | 子孙爻 | 我生者 |
| 妻财/金钱/效益 | 妻财爻 | 我克者 |
| 官非/疾病/丈夫/功名 | 官鬼爻 | 克我者 |

### 断卦步骤

1. **用神旺衰**：用神在月令（月建）和日辰（日建）下的状态
   - 生扶：旺相
   - 克制：衰弱
   - 月破：月令冲用神
   - 日破：日辰冲用神

2. **动爻分析**：动爻对用神的作用
   - 生用：有助
   - 克用：有阻
   - 化进/化退：力量变化

3. **变卦参考**：变卦反映事物发展方向

4. **六神参考**：
   - 青龙：喜庆、吉事
   - 朱雀：口舌、文书
   - 勾陈：拖延、纠结
   - 螣蛇：虚惊、怪异
   - 白虎：凶险、丧服
   - 玄武：暗昧、盗贼

### 旺衰判定

| 月令关系 | 状态 |
|---------|------|
| 用神与月令同五行 | 旺 |
| 月令生用神 | 相 |
| 用神生月令 | 休 |
| 用神克月令 | 囚 |
| 月令克用神 | 死 |

## 在咨询中的使用

### 事业决策辅助

1. 起卦后定用神为官鬼爻（功名/事业）
2. 看官鬼爻旺衰：旺则事业有利，衰则受阻
3. 看世爻与官鬼爻的关系：世爻持官鬼则自身能力与事业匹配
4. 动爻提示变化因素
5. 结合八字/紫微命理综合判断

### 婚恋占卜

1. 男占以妻财爻为用神，女占以官鬼爻为用神
2. 用神旺相：对方条件好或感情顺利
3. 用神生世爻：对方有意
4. 用神与世爻相冲：感情波折
5. 应爻（对方）与世爻（自己）的生克关系

### 失物/寻人

1. 以子孙爻或财爻为用神（视失物性质）
2. 用神所在宫位推断方位
3. 用神安静（不动）：物在近处
4. 用神发动：物已移动
5. 伏神：物被遮盖或藏匿

## 与梅花易数的区别

| 维度 | 六爻（ichingshifa） | 梅花易数（meihua-yishu） |
|------|---------------------|------------------------|
| 起卦方式 | 蓍草/铜钱/时间 | 时间/数字/声音/颜色 |
| 分析重点 | 纳甲六亲/世应/动爻 | 体用生克/卦象直读 |
| 适用场景 | 具体事项（失物/求财/官非） | 趋势判断/心境/方向 |
| 复杂度 | 较高（需排纳甲） | 较低（直接读卦象） |
| 时间精度 | 小时（时辰） | 秒级 |

建议：具体事项用六爻，趋势和方向用梅花。两者可互为补充。

## 注意事项

- 六爻卜卦为一次性占问，同一事不宜反复起卦（"初筮告，再三渎，渎则不告"）
- 起卦时需心诚专注，时间起卦需精确到时辰
- ichingshifa 库最后更新于 2023 年，功能已稳定但无新功能更新
- 依赖 sxtwl（天文历法库）进行农历计算，确保历法准确性
- 六爻断卦结果为文化参考，必须遵守 RULES.md 中的 disclaimer 和积极导向原则
- 不可用卜卦结果替代专业医疗/法律/财务建议
